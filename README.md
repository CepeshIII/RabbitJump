# Rabbit Jump 🐰🚀

[![Play on Itch.io](https://img.shields.io/badge/Play%20on-Itch.io-FA5C5C?style=for-the-badge&logo=itchdotio)](https://cepeshiii.itch.io/rabbit-jump)
[![Unity](https://img.shields.io/badge/Made%20with-Unity-black?style=for-the-badge&logo=unity)](https://unity.com)

**Rabbit Jump** is a fast-paced, addictive 2D platformer where you help a brave rabbit reach new heights. Leap through neon-lit cityscapes, dodge obstacles, and find your way back to your crashed spaceship!

## 🎮 Play the Game
The game is available to play directly in your browser or for download on Itch.io:
👉 **[https://cepeshiii.itch.io/rabbit-jump](https://cepeshiii.itch.io/rabbit-jump)**

## ⌨️ Controls
- **Arrow Keys / WASD:** Move Left and Right.
- **Touch Screen**

🛠️ Built With Unity 6000.0.60f1
